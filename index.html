Predicting Left Ventricular Ejection Fraction from Coronary Angiography

A deep learning framework combining convolutional and transformer-based modeling to estimate LVEF directly from invasive coronary angiography sequences.

Project Overview

This project explores automated prediction of Left Ventricular Ejection Fraction (LVEF) directly from invasive coronary angiography (ICA) videos.

The model combines convolutional feature extraction with transformer-based temporal modeling to capture both spatial structure and contrast-flow dynamics.
