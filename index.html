Predicting Left Ventricular Ejection Fraction from Coronary Angiography using 3D-CNN + Transformer

Supplemental materials for a deep learning framework that estimates LVEF directly from invasive coronary angiography sequences using spatiotemporal feature extraction and transformer-based temporal modeling.

Supplemental Research Summary

Project Overview

This project explores automated prediction of Left Ventricular Ejection Fraction (LVEF) directly from invasive coronary angiography (ICA) videos. The goal is to provide a faster and more accessible estimate of cardiac function from routinely collected angiographic sequences.

The proposed framework combines convolutional feature extraction with transformer-based temporal reasoning to capture both spatial structure and contrast-flow dynamics across frames.

Dataset

  • Dataset: Nanjing
  • Total videos: 1,041
  • Total patients: 413
  • Frame count: 13 to 114 frames per sequence
  • Image resolution: 1000×1000 to 512×512
  • Training split: 832 videos
  • Test split: 209 videos

Model Summary

The model uses a 2D+t convolutional backbone to extract spatiotemporal features from angiographic frames, followed by a transformer module to model temporal dependencies across the cardiac sequence.

Key Findings

  • The hybrid CNN + Transformer model performed substantially better than the transformer-only baseline.
  • The model achieved strong classification performance for identifying LVEF below 40%.
  • There is a tradeoff between accurately predicting low LVEF values and higher LVEF values.
  • Optimizing toward average predictions can reduce performance at the extremes of the LVEF range.

Future Work

  • Expand and refine the multi-view angiography framework.
  • Integrate vessel masks and segmentation-informed training.
  • Map segmented datasets more effectively to the main training dataset.
  • Improve generalization across different imaging conditions and views.
